# BeeSpeak
BeeSpeak is an AI-powered scam detection and engagement system that identifies fraudulent messages, assigns confidence scores, categorizes scam types, and extracts indicators like UPI IDs, bank details, and phishing links. It uses adaptive personas to safely engage scammers and gather intelligence.

[Demo Video]()

# Features
-  Real-time Scam identification and categorisation
-  Extraction of essential scam info (upi, link, etc)
-  Persona selection based on scam type
-  User selectable personas
-  Clean Json output for dashboard integration
